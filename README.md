## 📖 About This Repository

This repository contains my notes, commands, practical exercises, and mini projects while learning **Linux for DevOps**.

The goal of this repository is to build a strong foundation in Linux concepts that are essential for working with modern DevOps tools and cloud platforms.

---

## 🎯 Objectives

- Understand Linux fundamentals and architecture
- Learn Linux file system and navigation commands
- Manage files, permissions, users, and groups
- Understand process and package management
- Learn Linux networking concepts and commands
- Write basic Shell Scripts for automation
- Build mini projects using Linux commands and Bash scripting
- Prepare for DevOps and Cloud Engineering roles

---

## 🛠️ Technologies & Tools

- Linux (Ubuntu)
- Windows Subsystem for Linux (WSL)
- Bash Shell
- Git & GitHub
- Visual Studio Code
